Gebruik van For en While Loops

Wanneer gebruik je een For-loop?

Een for-loop gebruik je meestal wanneer je weet hoe vaak je de code binnen de loop wilt uitvoeren, bijvoorbeeld wanneer je door een lijst of reeks heen moet lopen. Dit maakt het geschikt voor herhalingen met een vast aantal iteraties.

Situatie 1: Lijst doorlopen

Een for-loop is handig wanneer je een lijst van items moet doorlopen, bijvoorbeeld een lijst van namen of producten. Je kunt elk element in de lijst afzonderlijk verwerken.

Situatie 2: Nummers afdrukken binnen een bereik

Een andere situatie is wanneer je een reeks van nummers wilt genereren en verwerken, bijvoorbeeld alle getallen van 1 tot en met 10. De for-loop maakt het gemakkelijk om deze reeks af te lopen zonder de iteratie handmatig bij te houden.

Wanneer gebruik je een While-loop?

Een while-loop gebruik je wanneer je niet precies weet hoe vaak de loop zal moeten herhalen. De herhaling gaat door zolang een bepaalde voorwaarde waar is. Dit maakt de while-loop geschikt voor situaties waar de conditie dynamisch kan veranderen.

Situatie 1: Gebruikersinvoer controleren

Een while-loop is nuttig wanneer je bijvoorbeeld wacht op geldige invoer van een gebruiker. Je blijft vragen om input tot de gebruiker een geldige waarde invoert.

Situatie 2: Willekeurig getal raden

Een andere situatie waarin je een while-loop zou gebruiken, is een spel waarin de gebruiker een willekeurig getal moet raden. De loop blijft doorgaan totdat het getal correct geraden is.
